GitHub+Hexo 搭建博客--我的第一篇博客

2019-11-22
  • hexo

首先恭喜我…终于搭建好博客了

​ 我来记录一下搭建的过程

工善其事 利先其器 下载所需工具

  1. 首先下载node.js 安装
  2. 下载 node.js'之后,cmd打开,执行命令 npm install hexo-cli -g 安装hexo, hexo官网
  3. 然后执行命令npm install hexo-deployer-git --save 安装部署到github的插件
  4. 选择一款你喜欢的主题 官方主题网址 https://hexo.io/themes/ 官方wiki网址 https://github.com/hexojs/hexo/wiki/Themes 然后使用git下载, 本网站使用了indigo

cmd走起 搭建博客

​ 搭配hexo官方文档使用

  1. 找一个空文件夹.打开cmd 使用如下命令

    hexo init 有可能clone 失败,也可能中途一些其他问题,请多次尝试,下载成功界面

    下载完成之后完整目录结构

  2. hexo 常用的几个命令

    • 清除缓存 hexo clean

    • 生成静态文件：hexo generate，简写hexo g

    • 启动服务器：hexo server，简写hexo s

    • 部署到远程站点： hexo deploy，简写hexo d

    • 写新文章: hexo n <title> 新文章在 source/_posts文件夹下面

      更多指令：https://hexo.io/zh-cn/docs/commands

    现在使用命令 hexo s 然后打开 http://localhost:4000/查看

定义文章链接,存放位置

hexo 新的文章位置默认在*source\_posts 文件夹下面,修改配置文件的Url部分,可以自定义文件夹

我的就是在_posts文件夹下面在用年份命名就可以这样配置

在hexo中插入图片

插入图片有两种方式,第一是在source文件夹下面建立一个文件夹用来存放图片,例如imgs,然后把图片插入md 中![](/imgs/picname.jpg)就可以显示.但是推荐第二种找到Writing,设置post_asset_folder为true

然后新建一个文章就可以发现在同名md文件夹下面会对应一个文件夹,把你的资源放在这个文件夹,然后设置 路径就可以了,例如插入一个1.jpg就直接写![](1.jpg)即可.

修改主题颜色,更新于2020.2.10 20:25

​ 这个主题颜色更改花了我好久好久,改的心力憔醉.主要还是跨行如跨山..

首先看作者github上面的issue,也就是说更改配置的cdn:false,让偶修改修改variable.less文件的配色就行了

​

可是修改了颜色我发现并没有效果,而且布局还乱了,这么就奇怪,我查了很久,认真看了其他的issue和作者的wiki,由于真的是什么都不太懂,作者写的这句话我都不太懂

​

后面我打开网站的时候发现他在访问一个unpkg.com的网站,去下载了一下东西,然后我打开网站下面的index.html发现了一些奇特的现象,(网站的样式由style.css决定)

​

到这里就懂了,设置为false的时候就是访问本地,难怪我改了如果为true,那么怎么修改都是没有乱用的.

到这里又有个问题,为什么到用本地的布局效果都不起做用了.我查了一下blog根目录中的public文件夹,发现其中的css,只有一个style.less(原本只是7kb,我这里已经转换了),再和theme中的主题css文件夹对比,发现就少了很多东西

​

​

那么问题就是生成的时候缺少这些css文件了???后来我试了拷贝过去也没有乱用,后来我看了作者wiki,提到了

然后我查看了hexo-renderer-less,原来这是个编译插件(我就这么叫吧😜),把less编译成css,然后我按照作者的说明安装了这个插件,然后在 hexo g之后就发现了,可以生产style.css文件了.接着就是修改颜色了.大多数颜色修改都是在variable.less文件中,但是如果想更多的自定义颜色,那么就要打开网页f12自己招标签修改了.我花了很长时间.这里小提一下各个颜色大概在哪里

文件名 设计颜色
variable.less 大部分颜色,文章crad颜色(在最下面)tags颜色,我们也可以学习,写全局变量控制颜色
layout.less 布局,比如首页的左边导航栏颜色等等
article.less 大多数文章的颜色
postless.less 页面导航(上一页下一页等等)(“阅读更多”)

还有很多,具体还是要打开浏览器然后f12查看然后自己在众多的less文件中查找相对应的关键字.

这里再说一下我使用了valine评论,修改这一块的颜色的时候,我发现只能改背景(对应就是pugins文件夹下面的valine.less),我想改里面文字颜色一直改不了,正当我要放弃的时候,我发现其实他也是下载了unpkg上的js代码,后来我顺着这个路径把这个代码下载下来,然后修改主题layout文件夹下面的variable.ejs

​

这里原来是指向一个网址,现在下载下来放入js文件夹下面,然后指定一下就可以了,代码中涉及一些颜色就直接改了.

这个就介绍到这里,累😴!!!
